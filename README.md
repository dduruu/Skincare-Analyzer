# Skincare-Analyzer
Python-based AI-powered skincare ingredient analyzer

### **Overview**
**Skincare-Analyzer** is a simple Python tool designed to decode complex skincare ingredient lists. It helps identify what is inside a product by using artificial intelligence to provide a clear report.

### **Key Features**
* **Simple Analysis:** Converts long and complex chemical names into a readable format.
* **Fast Results:** Uses the **Groq API** to get analysis results in seconds.
* **Detailed Info:** Provides a **safety score**, identifies **red flags**, lists **concerns**, highlights **key benefits**, and offers a helpful **summary** of the product.

### **How to Use**
1.  **Clone the repository** to your computer.
2.  **Install the library** by running:
    ```bash
    pip install groq
    ```
3.  **Add your API Key** to the code where it says `GROQ_API_KEY`.
4.  **Run the script** and paste your ingredients:
    ```bash
    python main.py
    ```
